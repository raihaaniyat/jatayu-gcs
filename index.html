What We Built

An end-to-end robotics platform that demonstrates two core capabilities of embodied intelligence: robotic arm manipulation driven by visual perception and language-guided reasoning, and LiDAR-based SLAM for real-time mapping and localization on a GPS-denied autonomous rover - all built on ROS 2.

Why It Matters

Most Vision-Language-Action models are tested only in simulation. The sim-to-real gap - sensor noise, calibration error, clutter, grasp instability - means actions that work digitally often fail on real hardware. Simultaneously, autonomous navigation in GPS-denied environments demands robust onboard mapping and localization. This project validates both capabilities on physical systems.

DEMONSTRATION 1 - ROBOTIC ARM MANIPULATION
Perception-to-Action Pipeline

The manipulation system captures the workspace using a ZED Mini stereo camera to produce aligned RGB and depth streams. Objects are detected and localized in 3D, then hand-eye calibration transforms camera-frame positions into the robot base frame. A VLA reasoning module interprets the scene and user commands, generating target poses that are translated into joint trajectories via MoveIt 2, driving the xArm to perform physical pick-and-place.

DEMONSTRATION 2 - LiDAR SLAM ON GPS-DENIED ROVER
Autonomous Mapping Pipeline

The rover operates in GPS-denied environments, relying entirely on onboard LiDAR for spatial awareness. Point cloud data is processed through the ROS 2 SLAM Toolbox to build real-time occupancy grid maps while simultaneously localizing the rover within the constructed map. This enables autonomous navigation without external positioning infrastructure.
